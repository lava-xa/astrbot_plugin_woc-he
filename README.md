# 群聊 @ 信息输出

这是一个 AstrBot 插件，用于监听白名单 QQ 群聊中的 `/woche @某人` 消息。

当白名单群内有人发送 `/woche @某人` 时，插件会先回复提示语，再回复被 @ 用户的 QQ 号、昵称、示例 IP、随机年龄、随机地址、随机手机号、随机学号和随机宿舍。

当白名单群内有新成员加入时，插件会按 `/woche` 相同格式发送该成员的信息。

## 配置

在插件配置中填写需要监听的群号：

```json
{
  "whitelist_group_ids": ["123456789"],
  "custom_output": "这里可以写每次回复时附加的自定义内容",
  "enable_member_join_info": true
}
```

只会处理白名单群聊中的 `/woche @某人` 消息和新成员入群事件，其他群聊或普通 @ 会被忽略。

`custom_output` 可以留空，留空时只输出 QQ 号和昵称。

`enable_member_join_info` 用于控制是否发送新成员入群信息，默认开启。
